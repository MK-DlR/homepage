notes:
    - remember the layout needs to be responsive!
        - avoid fixed width/height, instead use: 
            - max width
                - use fixed width for:
                    - small elements
                    - things that DO need a set size (sidebars, etc)
                - can also use static margin
            - max height*
                - avoid height: use margin/padding instead
    - flexbox is naturally responsive
        - flex-wrap
    - desktop / tablet / phone layouts
